# 06 · 人才与教育

> 这个行业最稀缺的不是钱，是**能同时搞定「比特（ML）」和「原子（硬件/控制）」的人**。
> 「年薪百万招不到人」是常态。本章看三件事：**人才从哪来、争夺有多疯、长期怎么补。**

---

具身智能的头部公司，几乎都是**顶尖高校实验室 + 大厂天才**直接孵化出来的。看懂血脉，就看懂了人才流向：

> 开枝散叶，几乎半个具身智能学术圈都与之相关。

## 二、人才争夺战：薪资数据（拨开迷雾的硬指标）

钱的流向不会说谎。2026 年的薪资数字，既证明人才之稀缺，也是泡沫的温度计：

| 层级 | 年薪 | 来源 |
|---|---|---|
| 行业平均招聘 | **33.3 万**（比 AI 行业均值 29.1 万高 14.6%） | 招聘平台 |
| 顶尖实验室应届博士（清北复交浙哈） | **60–70 万+** 起薪 | 行业报道 |
| 宇树「机器人控制算法专家」 | 70–100K×13 薪 ≈ **130 万** | 宇树招聘 |
| 具身算法工程师（专家岗） | 最高 **200 万** | 科锐国际薪酬指南 |
| 毕业一年的尖子 | 传 **300 万** | 36 氪 |
| 顶级科学家（挖角） | **千万级**（曾有 1.24 亿抢科学家的传闻） | 财经媒体 |

> 一句话：**「年薪百万招 0 工作经验工程师」已是真事。** 这是真实的稀缺，但当应届生喊到 300 万、
> 科学家论亿争抢时，它**同时也是泡沫的温度计**——薪资曲线的陡峭程度，和估值曲线一样需要打问号。

## 三、最稀缺的能力：bits × atoms 的复合型人才

为什么招不到人？因为这个行业要的能力**横跨两个长期割裂的世界**：

- **比特侧**：机器人学习（RL/模仿学习）、大模型/VLA、大规模 ML 系统。
- **原子侧**：机械设计、运动控制、嵌入式、硬件工程。
- **最稀缺 = 两者都懂的人**。纯算法的不懂硬件的脾气，纯机械的跟不上 VLA 的迭代。

**一个真实的人才迁移信号**：大量**自动驾驶（车端）人才正涌入具身**——「车端软硬协同」的经验
（端到端、数据闭环、量产降本）高度可迁移。智能驾驶的红利退潮，正好给具身输送了一批成熟工程师。

## 四、人才培养：长期管道刚刚搭起

人才荒倒逼教育端补课——**2026 年成为「具身智能学历教育」元年**：

- 教育部《普通高等学校本科专业目录（2026）》**正式新增「具身智能」本科专业**，
  首批批准 **9 所顶尖工科高校**开设；上海交通大学年招生约 **30 人**。
- 但本科培养周期 4 年起，**远水难解近渴**——未来 3–5 年人才缺口仍要靠高校实验室、车端迁移、海外回流填补。

## 五、判断（拨开迷雾）

1. **人才是真瓶颈**：复合型（bits×atoms）人才的稀缺是结构性的，不是发发 offer 就能解决，
   呼应供应链那条主线——**这个行业的命门，一半在精密制造，一半在复合型人才，都是「慢变量」。**
2. **薪资是泡沫温度计**：千万年薪、应届 300 万，与估值脱钩、Demo 注水是同一种过热的不同切面。
3. **得实验室者得天下**：头部公司=明星实验室外溢，下一批黑马大概率也会从今天的顶尖 Lab 里跑出来——
   **盯实验室，比盯公司更早看到未来。**
