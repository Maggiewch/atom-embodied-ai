# 智元机器人 AgiBot · 档案

> 中国全栈人形机器人头部玩家。靠「华为系硬核团队 + 极速量产 + 自研 GO-1 大脑 + 资本生态绑定」卡位，
> 走的是典型中国剧本：供应链整合 + 真实出货 + 政策加持。号称「国内最快破十亿营收的机器人公司」，
> 当前最敢喊量产数字、也最被质疑「营收质量与资本运作」的全栈玩家。

## 1. 创始团队与基因
智元是一支**华为系硬核团队**：CTO 彭志辉（稚晖君，1993 年生）出身华为「天才少年」计划，做过昇腾 AI 芯片与算法，
B 站千万粉的硬件大神，2022 年底离职、2023 年初创办智元——他给公司带来**极客式工程文化 + 流量号召力**。
董事长兼 CEO 邓泰华则是前华为副总裁，管过无线产品线与计算产品线，2025-03 接任法定代表人——他带来的是
**组织、供应链、资本运作的「成年人」能力**。

这套「天才少年 + 华为老将」组合，决定了智元的打法和 Figure 完全不同：**不靠单一明星叙事，而靠华为式
的「全产业链整合 + 极致工程节奏 + 生态绑定」**。这也是为什么智元能在三年内把出货推到万台量级——
它把华为做硬件的那套供应链与量产纪律，直接搬到了人形机器人上。

## 2. 技术路线
- **本体（三条产品线并行）**：远征系列全尺寸人形（A1 约 175cm、49+ 自由度；A2 约 169cm、40+ 自由度，
  单臂负载约 3kg，搭 ActionGPT 动作模型）；灵犀系列半尺寸人形（X2 约 131cm、约 35–37kg、27–31 主动自由度，
  模块化、主打交互/表演/陪伴）；精灵系列轮式双臂（G2 升级「十字腕」7 自由度力控臂、搭英伟达 Jetson Thor）。
  关键零部件如 PowerFlow 关节模组、灵巧手大量自研——这是它敢喊量产降本的底气。
- **大脑 GO-1（Genie Operator-1）**：2025-03 发布，号称「业界/国内首个通用具身基座大模型」。
  采用自研 **ViLLA 架构**（Vision-Language-**Latent**-Action）——比传统 VLA 多了「隐式动作」一层，
  靠预测 Latent Action Token 从**海量无标注人类视频**学习，再用 MoE（Latent Planner + Action Expert）落到真机动作，
  主打小样本快速泛化。这是其「全栈」标签的核心：本体 + 大脑都在自己手里。
- **数据策略**：2024 年底开源 **AgiBot World** 数据集（>100 万条真机轨迹、217 个任务、覆盖家庭/零售/工业/餐厅/办公
  五大场景），这是国内最大规模的开源真机数据之一，也是 GO-1 的训练底座。数据飞轮是否真正转起来仍待验证，
  但开源数据集本身已成行业事实标准之一。

## 4. 商业模式与客户
- **怎么赚钱**：以**卖本体**为主（三条产品线覆盖工业/商用/科教/文娱），叠加模型与二次开发生态。
  与 Figure「估值远超收入」不同，智元已是行业里**少数有真实规模化出货 + 十亿级营收**的玩家。
- **真实客户与订单**：中国移动（200 台、¥7,800 万）、富临精工（近百台 A2-W）、龙旗科技（数亿元、近千台精灵 G2，
  号称国内工业具身最大单)。**注意**：龙旗、卧龙等多家下单方同时也是智元的产业股东/被投企业——
  「股东即客户」的关联交易成色，是判断营收质量的关键。

生态内既是供应商、又是客户、又是股东，营收与订单的「自循环」成分需要警惕。

## 7. 拨开迷雾（争议与真实进度）
- **营收质量 vs 营收规模**：¥10.5 亿营收是行业里最实的之一，但需拆开看——多少来自**关联股东下单**（龙旗、卧龙等）、
  多少是真正市场化外部订单？「股东即客户」的闭环会让营收数字好看，却未必反映真实市场需求。**这是最该死磕的一点。**
- **「万台」是下线数，不是在役/创收数**：万台下线≠万台稳定作业、≠万台付费部署。其中相当比例是文娱/讲解/科教等
  **低作业含量**形态（灵犀表演、远征讲解），与「干活的劳动力」是两回事。
- **自主程度普遍偏低**：当前真实落地以**半自主 + 遥操作 + 演示**为主，GO-1 的端到端自主作业仍在爬坡。
  Demo（叠衣、泛化操作）惊艳，但「一次成功的演示」与「产线稳定自主」之间差距巨大。
- **资本运作疑云**：21 亿控股上纬新材、否认借壳但实际控制人变更为邓泰华——这是**A 股「类借壳」资本术**，
  上纬股价 16 天 11 个涨停的暴涨，更多是壳价值与情绪，而非业务协同兑现。叠加港股 IPO 与 ¥500 亿估值传闻，
  资本动作的密集度已超过业务本身。
- **可持续性**：「358 宏图」（2030 破千亿）是极其激进的叙事；当前靠产业资本持续输血 + 关联生态托底。
  若外部市场化需求跟不上、或 IPO 兑现不及预期，估值与营收增速都有回撤风险。
