# 10 · 资本与投资人视角

> 钱从哪来、聪明钱怎么想。这一章看的不是公司，而是**站在公司背后的资本**——
> 谁在下注、押的是哪条逻辑、他们的世界观如何塑造了这个行业的走向。

商业模式（`04-`）回答「公司怎么赚钱」；可持续性（`08-`）回答「钱够不够烧」；
本章回答上游的问题：**「聪明钱为什么愿意给钱，他们在赌什么」**。
