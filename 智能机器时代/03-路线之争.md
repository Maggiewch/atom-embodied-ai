# 03 · 路线之争：VLA 之外，还有谁在跑

> 起因：「何小鹏放弃了 VLA 路线」的传闻。核实后发现——**这是一个被压缩失真的版本。**
> 但它恰好掀开了一场真实的路线之争。

---

## 核实：他没有「放弃 VLA」，而是放弃了「第一代 VLA」

| | 传闻版本 | 真实情况 |
|---|---|---|
| 放弃了什么 | 「放弃 VLA 路线」 | 放弃**标准/第一代 VLA 架构**（语言转译 + 监督学习） |
| 改成了什么 | 「改了研究方向」 | 做了**第二代 VLA**——仍叫 VLA，只是重构 |
| 语境 | 像在说具身机器人 | 主要是**智能驾驶**，但小鹏刻意把它接到具身智能 |

**准确说法**：2025-11-05 小鹏科技日发布**第二代 VLA**，确实「停止传统 VLA 研发、投入 20 多亿元探索新方向」。所以传闻不是空穴来风，但「放弃 VLA」是把「拆掉旧架构、做新 VLA」误读成了「抛弃 VLA」。他是**在 VLA 上加倍下注**。

**第二代砍掉了两根拐杖**：
1. **拆掉语言中间转译层**（V→L→A 变 V+L→A）：语言从「理解枢纽」降级为「辅助输入」，减少信息损耗。
2. **拆掉监督标注**（监督 → 自监督）：于是能吃下近 1 亿条视频片段。执行层并行挂上**世界模拟 + 强化学习 + 轨迹规划**。

> 对照看：这正是对**数据瓶颈**的正面进攻——语言转译和人工标注都是扩数据的枷锁，他全砍了。不是退却，是为 scaling 做的架构手术。

---

## 产业站队（智驾语境）

- **VLA 派**：小鹏、理想（郎咸朋：「VLA 就是自动驾驶最好的模型方案」，用 GPT 的方式做）。
- **世界模型派**：华为、蔚来。
- **怀疑派**：宇树王兴兴直言「VLA 是相对傻瓜的架构」（更偏控制 / 强化学习）。

---

## 真正的并行算法范式（具身智能语境）

1. **VLA（模仿学习 + 语言可控基础模型）**——当前主流前沿。π0、OpenVLA、RDT、GR00T、Helix。赌「把互联网知识转成机器控制 + 泛化长尾」。
2. **世界模型 / 基于模型**——先学能预测世界的模型，再规划或 RL。Genie、Cosmos、DreamerV3、1X 世界模型。赌「先懂物理再行动」。**LeCun 一派反对语言当物理智能基底——小鹏「拆掉语言拐杖」哲学上其实站到了这边。**
3. **强化学习 / sim-to-real**——大规模并行仿真 + 域随机化。在**腿足运动**上成果最硬（宇树、ANYmal、波士顿动力）。
4. **扩散 / 流匹配策略**——Diffusion Policy、flow-matching（π0 用流匹配）。是「动作表示」之争（自回归 token vs 扩散 vs 流）。
5. **自监督 / 互联网视频预训练**——从无动作标签视频抽取动作知识。V-JEPA、LAPA、Genie latent action。**破解数据瓶颈的路。**
6. **经典模块化控制 + 学习混合**——感知→规划→控制老栈 + MPC / 全身控制再用学习增强。

---

## 最该记住的一点：这些路线在融合，不是互斥

「放弃 VLA」式框架最大的误导，是把它讲成你死我活。实际上**前沿在杂交**——小鹏二代 VLA 字面上就是「VLA 躯干 + 世界模拟 + 强化学习」缝在一起；world model 与 VLA 的优化方向在趋同。

> 真正的判断方式：别问「它属于哪一派」，问「它把数据效率、泛化和那几个『9』往前推了多少」。
