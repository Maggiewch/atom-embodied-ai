# 04 · 商业地图：能赚钱的 5 个 vs 最有想象力的 5 个

> 一把尺子贯穿：**商业价值 = 已在算平账、出营收；想象力 = 上限极高，但卡在可靠性 / 成本 / 安全几道坎后面。**

---

## 现在最具商业价值的 5 个场景（已在赚钱或正跨过盈亏线）

1. **仓储物流操作：件级分拣 / 拆码垛 / 卸柜**。混放抓取、卸随机堆叠纸箱。岗位脏、工伤高、招不满、可 7×24。**P&L 最清楚转正的一类**（Amazon 收 Covariant、Dexterity、BD Stretch、Symbotic）。
2. **制造业柔性上下料 / bin picking / 机床看护**。从筐里抓杂乱金属件、给 CNC 上下料、黑灯生产（Mujin、Photoneo）。
3. **农业：采摘 / 除草 / 分级**。逻辑不是「比工人便宜」，是「根本招不到季节工」；果实有时效、每株不同（Carbon Robotics、Advanced Farm）。
4. **食品加工 / 预制餐装配**。处理会变形、每件不同的物料；劳动力极度短缺（Chef Robotics 有真实 RaaS 部署）。
5. **餐饮 / 酒店 / 医院 移动配送与服务**。**单位部署量最大**，国内尤甚（普渡、擎朗）。**诚实提醒**：这类「智能」更多是导航而非灵巧操作——商业上最成功，算法上却是具身 AI 偏「易」的一端。

> **共同铁律**：任务以前做不了（非结构化），或人根本招不到，**且不要求极致灵巧**。越接近结构化、越不需要灵巧手，越早赚钱。

---

## 最具想象力的 5 个场景（上限惊人，仍在坎后）

1. **通用人形家庭机器人**。家是终极非结构化环境，TAM 是每个家庭，「机器人的 iPhone 时刻」。卡在成本、可靠性、人身边安全——**今天炫酷视频大多是遥操作**（Figure、Optimus、1X、宇树）。
2. **老龄化社会的具身护理**。可能是**经济意义最大的梦**（中日欧刚需）。卡在直接接触人体、安全 + 灵巧 + 情感三重难关（日本 Robear 试了十几年）。
3. **自主手术 / 自主医疗操作**。卡在监管、责任、极端安全；STAR 自主缝合只是早期火花。
4. **太空 / 灾难 / 核 等极端环境作业**。价值是「人命」，独一无二。卡在极端可靠性、通信时延、地形不可预测。
5. **自驱动实验室 + 柔性微工厂**。具身 AI 当「智能的物理执行层」（伯克利 A-Lab 做材料发现）。卡在通用性。串起这 5 个梦的底层能力，都是同一个**长程、开放任务的自主执行**——误差指数级累积，这是研究最前沿。

---

## 怎么看这波浪潮 + 商业化希望

> **总判断：这是一次真实的能力拐点，但它会以「工业自动化的老方式」落地——窄、垂直、ROI 驱动、不性感——而不是「人形 AGI 叙事」承诺的方式（通用家庭机器人很快到来）。**

- **真在卷什么**：「基础模型」配方第一次进了机器人；机器人白嫖了整个 LLM 基建。资本赌它复制 LLM 曲线——**这个假设远没被证明，泡沫就在这里。**
- **整个浪潮压在一个赌注上**：动作能不能像语言一样 scaling？机器人没有「动作的互联网」。全行业在赛跑解决**数据瓶颈**（遥操作 / 仿真 / 视频 / 跨本体，各有死穴）。**谁解决可规模化的动作数据，谁赢——这比模型本身更重要。**
- **护城河不在模型**（模型在收敛、开源）：在 ① 数据飞轮 / 部署入口，② 可靠性工程（抠那几个「9」），③ 硬件成本，④ RaaS 业务能力。**最先把机器人跑进真实运营的玩家，靠数据复利越拉越开。**
- **风险**：资本期待 LLM 速度的回报，物理世界给不了 → 一次预期回调几乎必然。真问题是那些**又真又闷的垂直生意**能否撑过寒冬，别把婴儿和洗澡水一起泼掉。
- **看多信号**：动作数据 scaling 出现明确曲线、几个垂直 RaaS 跑到规模盈利。**看空信号**：融资估值继续靠人形演示驱动，而真实部署的成功率与单位经济不动。
